## 프로젝트 실행 방법

- node 16 버전 이상
- 프로젝트 root 경로에서 npm install로 패키지 다운로드
- npm run start 실행 (localhost:3000)

## 사용 라이브러리

- ant-design (UI 라이브러리)
- tailwind (CSS 라이브러리)
- typescript (타입스크립트)
- axios (REST API 통신 라이브러리)
- query-string (URL 정보)

## 디렉토리 구조

- /components : 컴포넌트
- /pages : UI 페이지
- /service : api 호출
- /type : tpyescript type 폴더
- /util : 공통 함수 폴더
- 실무에서는 각 폴더에 프로젝트 메뉴나, 도메인에 따라 세분화 하여 관리합니다.
